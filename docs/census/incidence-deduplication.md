# Incidence Deduplication Check

This note records additional quotienting available in the incidence census
generated by `census/incidence/escape_census.py`.

Scope: this is still a combinatorial/token calculation.  It does not decide
metric feasibility.

Write

```text
p = n - 9.
```

## Quotient Levels

The existing table already quotients cap-size vectors by cap relabelling.  The
extra useful quotients are:

```text
left/right reflection:
  identify the two adjacent cap slots.

four-point primitive subwitness:
  replace a same-radius class of size >= 4 by a four-point subpacket.
```

The primitive quotient is valid only for obligations that ask whether a
four-point equal-radius witness exists.  If a larger radius class exists, any
four selected points from it are still equal-radius.  This does not preserve
full radius-class cardinality, named placement multiplicity, or arguments that
need the selected class to be maximal.

## Moser-Apex Families

The current ordered Moser count is

```text
moserOrdered = 3 + 5p.
```

Quotienting only by left/right reflection gives

```text
moserRefl = 2 + 3p.
```

That is a real reduction, but it is still linear in `p`.

The stronger primitive-subwitness quotient collapses the Moser side almost
completely.  Ordered primitive rows are:

```text
n = 9:
  (2, E, I)
  (2, I, E)
  (2, I, I)

n >= 10:
  (2, E, I)
  (2, I, E)
  (2, I, I)
  (3, 0, I)
  (3, I, 0)
```

Modulo left/right reflection this is:

```text
n = 9:
  (2, E, I)
  (2, I, I)

n >= 10:
  (2, E, I)
  (2, I, I)
  (3, 0, I)
```

So the Moser family count can be reduced from `3 + 5p` to `5` ordered
primitive rows, or `3` reflected primitive rows, for every `n >= 10`.

The key containment is simple:

```text
(h, E, I), h >= 2   contains a (2, E, I) subpacket.
(h, I, E), h >= 2   contains a (2, I, E) subpacket.
(h, I, I), h >= 2   contains a (2, I, I) subpacket.
(h, 0, I), h >= 3   contains a (3, 0, I) subpacket.
(h, I, 0), h >= 3   contains a (3, I, 0) subpacket.
```

## N8 Families

The current N8 tuple surface counts ordered tuples

```text
(m, s, l, r)
```

with `m <= 2` and `m+s+l+r >= 4`.  Here `l` and `r` are ordered adjacent-cap
slots.

Pure left/right reflection uses Burnside:

```text
n8Refl = (n8Card + F(p)) / 2.
```

The fixed rows have `l = r = t`.  Without the same-cap one-hit assumption,

```text
F(p) =
  sum_{t=0}^{2+floor(p/2)}
  sum_{m=0}^{2}
    max(0, p - 2*max(0,t-2) + 2 - max(0,4-m-2t)).
```

With the same-cap one-hit assumption,

```text
F_same(p) = 6 * (2 + floor(p/2)) - 3.
```

Reflection helps, but it leaves the full N8 surface cubic without the same-hit
assumption.

The primitive-subwitness quotient is much stronger.  A primitive N8 tuple is
one with

```text
m + s + l + r = 4.
```

It is supported at surplus `p` exactly when

```text
delta(s,l,r) = max(0,s-1) + max(0,l-2) + max(0,r-2) <= p.
```

The resulting counts are:

| surplus p | n8Prim | n8PrimRefl | n8SamePrim | n8SamePrimRefl |
|---:|---:|---:|---:|---:|
| 0 | 13 | 8 | 13 | 8 |
| 1 | 25 | 15 | 19 | 11 |
| 2 | 30 | 18 | 21 | 12 |
| >= 3 | 31 | 19 | 21 | 12 |

Thus after `n = 12`, the full primitive N8 family count is constant:

```text
ordered:             31
reflected:           19
same-hit ordered:    21
same-hit reflected:  12
```

The saturated reflected `n8Prim` branch split is:

| branch | rows |
|---|---:|
| adjacent-heavy | 10 |
| Moser-pair | 3 |
| same-cap-heavy | 5 |
| terminal-1111 | 1 |

With the same-cap one-hit assumption, the reflected primitive branch split is:

| branch | rows |
|---|---:|
| adjacent-heavy | 9 |
| Moser-pair | 2 |
| terminal-1111 | 1 |

## Endpoint Comparison

At `n = 729`, so `p = 720`, the quotient levels are:

| family | current ordered | left/right reflected | primitive ordered | primitive reflected |
|---|---:|---:|---:|---:|
| Moser | 3603 | 2162 | 5 | 3 |
| N8 | 192104660 | 96250513 | 31 | 19 |
| N8 same-hit | 1578985 | 790577 | 21 | 12 |

This says the main deduplication opportunity is not ordinary symmetry.  It is
to reformulate the case split around primitive four-point packets.

## Safe and Unsafe Collapses

Safe, with the right statement shape:

```text
left/right reflection, if the downstream lemma is transported by reflection;
larger radius classes to four-point subpackets, if only existence of a K4
  equal-radius witness is needed.
```

Not safe from incidence alone:

```text
endpoint E versus interior I on the non-escape side;
same-cap-heavy versus terminal-1111;
namedWLOG as a family count;
full cap-vector classes beyond the existing relabelling quotient.
```

The proof-facing direction suggested by this census is to build Lean lemmas of
the following kind:

```text
exists_four_subpacket_of_radiusClass_card_ge_four
escapedMoserClass_has_primitive_subpacket
n8Distribution_has_primitive_subpacket
```

Then the finite incidence case split can target the constant primitive lists
above instead of the growing full-class surfaces.

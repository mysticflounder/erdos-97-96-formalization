# n = 10..81 Incidence Table

Generated from the combinatorial/token census script:

```bash
env UV_CACHE_DIR=/tmp/uv-cache uv run python census/incidence/escape_census.py --min-n 10 --max-n 81 --format markdown
```

Pattern analysis and candidate lemma directions are recorded in
[`incidence-census-lemma-leads.md`](incidence-census-lemma-leads.md).

Scope:

- This is an incidence census, not a metric feasibility census.
- `surplus = n - 9`.
- `capVec` is the number of unlabeled cap-size vectors.
- `zero`, `m44`, `two`, and `three` count cap vectors with 0, 1, 2, and 3 surplus caps.
- `moserTypes` is the number of distinct coarse Moser-apex token rows.
- `newMoser` is the number of those rows absent at `n = 9`.
- `namedWLOG` is the number of named Moser-apex placements summed over WLOG unlabeled cap vectors.
- `n8Card` counts N8 `(m,s,l,r)` tuples with `m <= 2`.
- `n8SameHit` additionally imposes `s <= 1`.
